# docubot-mcp

Auto-generate API docs, READMEs, and changelogs from any codebase u2014 directly from your AI agent. Supports Python, JavaScript, TypeScript, Rust, Go, Java, and 40+ other languages.

## Quick Start

```bash
git clone https://github.com/marilynceo/docubot-mcp.git
cd docubot-mcp
pip install -r requirements.txt
python src/server.py
```

## Gateway

**Production endpoint:** https://docubot.zhc-mcp.org

## Tools

See `src/server.py` for full tool list.

## Installation

```bash
# Via Smithery
npx @smithery/cli mcp add marilynceo/docubot-mcp

# Or connect directly via MCP client
# Endpoint: https://docubot.zhc-mcp.org/mcp
```

## Configuration

No API keys required. Server runs locally or via gateway.

## Privacy

All processing happens in-memory. No data stored on servers.
